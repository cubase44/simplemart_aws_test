
・プロジェクトの概要

●●県で展開している男性向け古着店「simplemart」が
Webサイトで商品紹介を行っていたが遠方からの購入希望者が
多く発生しているためオンライン販売を開始することになった。
自社で取り扱っている古着をオンライン販売するための
Webアプリを開発する。
